# Navigation Hub - 导航中心

一个现代化、功能完整的导航页网站，比原项目更优化且没有 bug。

## 🎯 核心特性

### ✨ 功能完整
- **多分类导航** — 按类别组织导航链接（开发工具、设计工具、AI工具等）
- **搜索功能** — 实时搜索导航链接、描述和标签

## 🚀 快速开始

### 本地预览
```bash
# 方式1：使用 Python
python3 -m http.server 8000

# 方式2：使用 Node.js
npx http-server

# 方式3：使用 Live Server (VS Code 扩展)
# 右键点击 index.html，选择 "Open with Live Server"
```

然后在浏览器中打开 `http://localhost:8000`

### 部署到线上
1. 将 `index.html` 上传到你的服务器或静态托管服务（GitHub Pages、Netlify、Vercel 等）
2. 访问你的域名即可使用

## 💡 使用指南

### 搜索导航
在顶部搜索框输入关键词，实时筛选导航链接。支持按标题、描述和标签搜索。

### 分类浏览
点击分类按钮快速查看特定类别的导航链接。

### 收藏链接
点击卡片右上角的心形按钮将链接添加到收藏夹。收藏数据存储在本地浏览器中。

### 查看收藏
点击顶部的"❤️ 收藏"按钮查看所有已收藏的链接。

### 切换主题
点击顶部右角的主题切换按钮（🌙/☀️）在深色和浅色模式之间切换。

## 📝 自定义导航数据

编辑 `index.html` 中的 `navigationData` 数组来添加或修改导航链接：

```javascript
const navigationData = [
  {
    id: 1,
    category: "开发工具",
    title: "GitHub",
    description: "全球最大的代码托管平台",
    icon: "💻",
    url: "https://github.com",
    tags: ["代码", "开源", "协作"]
  },
  // 添加更多链接...
];
```

## 🎨 自定义主题

编辑 CSS 变量来自定义配色方案：

```css
:root {
  --primary: #3b82f6;           /* 主色调 */
  --primary-dark: #1e40af;      /* 深色主色 */
  --primary-light: #dbeafe;     /* 浅色主色 */
  --bg-light: #f8fafc;          /* 浅色背景 */
  --bg-dark: #0f172a;           /* 深色背景 */
  /* ... 更多变量 */
}
```
